Red Hat GitHub Actions Demo

Leverage OpenShift Developer Tools and GitHub Actions

Red Hat GitHub Actions Workflow is designed to help you work seamlessly with Red Hat Developer tools and Openshift clusters.

This application used the RedHat Starter Workflow to get an app running in minutes.

It's awesome

4 Easy Steps

  1. Get a sandbox cluster https://developers.redhat.com/developer-sandbox
  2. Add the workflow from https://developers.redhat.com/developer-sandbox
  3. Add the secrets for your openshift cluster and default container registry.
  4. Push code, and see this application running in minutes
